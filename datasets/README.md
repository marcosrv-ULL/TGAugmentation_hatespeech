# Datasets

This directory is intended to hold the data used in the paper **"From slurs to slots: LLM masking and Telephone-Game Augmentation for multiclass hate speech detection"**.

## ⚠️ Important Note on Data Availability

To adhere to safety guidelines and prevent the dissemination of harmful or hateful content on the web, **we have not included the raw or augmented text datasets in this public repository.**

Currently, **only the lexicon is included** (`lexicon/lexicon.jsonl`). This resource contains masked forms, placeholder codes, and aggregated counts, ensuring that no explicit hateful strings are exposed.

The following datasets are missing from this public directory:
* `ethos_augmented/`
* `supertweeteval_augmented/`
* Our custom-collected dataset.

### How to request the data

The full datasets are available to the research community upon request. If you wish to access them to reproduce our experiments or for your own research, please contact the authors at:

📧 **mrodrive@ull.edu.es**

We will respond as quickly as possible with the necessary files and instructions on how to place them in this folder to run the code successfully.
